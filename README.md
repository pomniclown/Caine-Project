# Caine Alpha 1.0

Короче, это мой проект по созданию нормального ИИ Кейна из "The Amazing Digital Circus". Я задолбался смотреть на ботов, которые пишут две строчки и тупят, поэтому решил обучить свою модель на базе Llama 3.

### В чем прикол этого проекта?
Я пытаюсь выжать из модели максимум экспрессии. Чтобы он не просто отвечал "Привет", а реально вел себя как поехавший конферансье. 

**Что уже сделал в Alpha v3:**
* **Вылечил "похмелье":** Раньше модель зацикливалась на слове "глоинки" и несла чушь. Сейчас всё стабильно, отвечает логично, но в своем стиле.
* **Масштаб:** Настроил так, чтобы он мог выдать огромный текст (до 10к токенов), если его разговорить. 
* **Анти-попугай:** Поставил жесткие штрафы за повторы. Теперь Кейн реально импровизирует.

### Как это запустить (если интересно)
Я использую **Unsloth** для обучения в Колабе и потом запекаю всё в **GGUF**. 
Если хочешь потестить у себя в LM Studio, ставь такие настройки:
* **Температура:** 0.9 (чтобы не был скучным).
* **Repeat Penalty:** 1.2 (чтобы не повторялся).
* **Контекст:** 4000 (оптимально для Альфы, чтоб модель не тупила).
